# UI/UX Design Guide & Wireframes

## Overview
This document provides visual design guidelines and wireframe descriptions for the Builder App interface.

---

## Design Principles

### 1. User-Centric Design
- **Intuitive Navigation**: Users should find features without training
- **Progressive Disclosure**: Show advanced features only when needed
- **Contextual Help**: Provide help where users need it

### 2. Visual Hierarchy
- **Clear Typography**: Use size and weight to establish importance
- **Strategic Color Use**: Guide attention to important actions
- **Whitespace**: Prevent cognitive overload

### 3. Consistency
- **Uniform Components**: Same elements behave the same way
- **Predictable Patterns**: Users learn once, apply everywhere
- **Brand Cohesion**: Consistent visual language throughout

### 4. Accessibility
- **High Contrast**: Readable text on all backgrounds
- **Large Touch Targets**: 44x44px minimum for mobile
- **Screen Reader Support**: Proper ARIA labels
- **Keyboard Navigation**: Full keyboard accessibility

---

## Color Palette

### Primary Colors
```
Primary Blue: #2563EB (Interactive elements, CTAs)
Primary Dark: #1E40AF (Hover states)
Primary Light: #DBEAFE (Backgrounds, highlights)
```

### Secondary Colors
```
Success Green: #10B981 (Confirmations, completed items)
Warning Orange: #F59E0B (Cautions, pending items)
Error Red: #EF4444 (Errors, critical alerts)
Info Blue: #3B82F6 (Information, help text)
```

### Neutral Colors
```
Gray 900: #111827 (Primary text)
Gray 700: #374151 (Secondary text)
Gray 500: #6B7280 (Tertiary text, placeholders)
Gray 300: #D1D5DB (Borders, dividers)
Gray 100: #F3F4F6 (Backgrounds)
Gray 50: #F9FAFB (Light backgrounds)
White: #FFFFFF (Card backgrounds)
```

---

## Typography

### Font Families
```
Primary: 'Inter', -apple-system, BlinkMacSystemFont, sans-serif
Monospace: 'Roboto Mono', monospace (for dimensions, measurements)
```

### Font Scales
```
Display: 48px / 60px line-height (Hero sections)
H1: 36px / 44px (Page titles)
H2: 30px / 38px (Section headers)
H3: 24px / 32px (Subsection headers)
H4: 20px / 28px (Card titles)
H5: 18px / 26px (Small headers)
Body Large: 16px / 24px (Primary content)
Body: 14px / 20px (Secondary content)
Body Small: 12px / 18px (Captions, labels)
```

## Animation Guidelines

### 1. Transitions
- **Page transitions**: 200ms ease-in-out
- **Modal open/close**: 300ms ease-out
- **Dropdown menus**: 150ms ease-in
- **Hover effects**: 100ms ease-out

### 2. Microinteractions
- **Button press**: Scale down 98% on active
- **Card hover**: Lift with 4px shadow
- **Toggle switches**: Slide 200ms
- **Checkboxes**: Checkmark appears with bounce

### 3. Loading Animations
- **Skeleton screens**: Shimmer effect 1.5s loop
- **Spinners**: Rotate 360Â° in 1s
- **Progress bars**: Smooth fill animation
